Protocole d'intégration asynchrone pour la passerelle d'inférence Axis

1. Fonctionnement Général

Axis agit comme un relais asynchrone vers l'API Kaggle afin d'exécuter des modèles lourds sans bloquer les connexions HTTP des applications clientes. Cela permet de s'affranchir des limitations de durée de vie des architectures serverless (comme les Edge Functions de Supabase limitées à quelques dizaines de secondes).

Toute intégration doit suivre le protocole en deux étapes :

  1. Initialisation : Envoi de la requête de prompt (format compatible OpenAI) pour créer une tâche en file d'attente.
  2. Polling (Boucle de vérification) : Interrogation récurrente (toutes les 3 à 5 secondes) de l'API jusqu'à obtention du statut final (complétion ou échec).

2. Initialisation d'une tâche

Pour soumettre un prompt d'inférence au modèle.

POST /v1/chat/completions

En-têtes requis :

Authorization: Bearer YOUR_AXIS_API_KEY
Content-Type: application/json

Corps de la requête (Body JSON) :

{
  "model": "axis-pro",
  "messages": [
    {
      "role": "user",
      "content": "Écris une fonction de tri en Python."
    }
  ]
}

Réponse (HTTP 202 Accepted) :

{
  "status": "queued",
  "job_id": "9b1deb4d-3b7d-4bad-9bdd-2b0d7b3dcb6d",
  "check_status_url": "http://localhost:3000/v1/jobs/status?id=9b1deb4d-3b7d-4bad-9bdd-2b0d7b3dcb6d"
}

3. Récupération du statut (Polling)

Appelé récursivement par l'application cliente.

GET /v1/jobs/status?id=YOUR_JOB_ID

Réponse (Si le traitement est en cours - HTTP 200 OK) :

{
  "job_id": "9b1deb4d-3b7d-4bad-9bdd-2b0d7b3dcb6d",
  "status": "processing"
}

Réponse (Si terminé avec succès - HTTP 200 OK) :

{
  "job_id": "9b1deb4d-3b7d-4bad-9bdd-2b0d7b3dcb6d",
  "status": "completed",
  "choices": [
    {
      "message": {
        "role": "assistant",
        "content": "Voici votre fonction de tri en Python..."
      }
    }
  ]
}

4. Exemple d'implémentation en Python

Voici un script Python simple intégrant la boucle de polling.

import requests
import time

AXIS_API_KEY = "votre_cle_axis"
BASE_URL = "http://localhost:3000"

headers = {
    "Authorization": f"Bearer {AXIS_API_KEY}",
    "Content-Type": "application/json"
}

# 1. Envoi du prompt
payload = {
    "model": "axis-pro",
    "messages": [{"role": "user", "content": "Hello World!"}]
}

response = requests.post(f"{BASE_URL}/v1/chat/completions", json=payload, headers=headers)
data = response.json()
job_id = data["job_id"]
print(f"Tâche initialisée : {job_id}")

# 2. Boucle de Polling
while True:
    time.sleep(3)
    status_response = requests.get(f"{BASE_URL}/v1/jobs/status?id={job_id}", headers=headers)
    status_data = status_response.json()
    
    status = status_data["status"]
    print(f"Statut : {status}")
    
    if status == "completed":
        content = status_data["choices"][0]["message"]["content"]
        print(f"\nRéponse du modèle :\n{content}")
        break
    elif status == "failed":
        print(f"Erreur : {status_data['error']}")
        break
